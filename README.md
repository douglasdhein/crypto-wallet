# Crypto Wallet

Crypto Wallet e uma aplicacao web para acompanhar criptomoedas com uma interface simples, moderna e focada em visualizacao de dados.

A proposta do projeto e exibir informacoes de mercado, como principais criptomoedas, ativos em tendencia, precos, variacao em 24 horas, market cap e graficos resumidos.

## Tecnologias

O projeto esta sendo desenvolvido com:

- Next.js
- TypeScript
- Tailwind CSS
- Node.js
- MongoDB
- CoinMarketCap API

## Como rodar localmente

Antes de iniciar, tenha o Node.js instalado na maquina.

Instale as dependencias:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse a aplicacao no navegador:

```txt
http://localhost:3000
```
